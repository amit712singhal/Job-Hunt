# Career Link

Career Link is a web application designed for hirers and recruiters to connect and manage job postings and applications efficiently.

## Installation Instructions

1. **Clone the repository:**

   ```cmd
   git clone https://github.com/amit712singhal/career-link.git
   cd career-link
   ```
